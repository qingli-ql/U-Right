# Electron UI 重构执行清单

更新时间：2026-04-03

这份文档用于把 Electron Host UI 从“网页式后台”逐步收口为“macOS 工具型界面”。

## 目标

- 降低卡片层级和容器噪音
- 提高桌面端信息密度与扫描效率
- 让设置页、菜单编辑页更偏向专业工具而不是展示面板
- 收口命名、状态语义、文案层级
- 为后续 dark mode、组件拆分和可维护性打底

## 执行顺序

- [x] 第 0 步：落档重构计划，建立执行和标记机制
- [ ] 第 1 步：全局骨架收口（进行中）
  - 收紧 `Shell / Sidebar / Header / Footer`
  - 降低 Hero 和大卡片存在感
  - 建立更明确的状态语义和更轻的保存反馈
- [ ] 第 2 步：Settings 普通页密度提升
  - `General / Apps / AI / Advanced`
  - 优先列表、表单、分隔线，不继续堆卡片
- [ ] 第 3 步：Detected Apps 页面重构
  - 改为标准桌面列表/表格视图
  - 提升横向比较与诊断效率
- [ ] 第 4 步：Context Menu 工作台结构重构
  - 强化三栏编辑器结构
  - 收起开发态信息
  - 让预览更像真实 Finder 菜单
- [ ] 第 5 步：结果页、日志页、引导页语义统一
  - 降低展示感
  - 提高可读性和工具感
- [ ] 第 6 步：命名审计与文案收口
  - 导航、页面、模块、按钮命名统一
  - 清理中英文混用和开发态外露
- [ ] 第 7 步：组件与 token 收口
  - 收敛颜色、间距、圆角、控件高度
  - 为后续拆分 `App.tsx` 做准备
