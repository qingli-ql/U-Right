**导出 API**
- `refreshFinderWindows(options)`：刷新 Finder 窗口，支持按目录优先刷新相关窗口，定义见 `electron/src/main/finder-view-actions.ts:167`
- `toggleFinderHiddenFiles(options)`：切换指定目录场景下的隐藏文件显示，定义见 `electron/src/main/finder-view-actions.ts:208`
- 相关类型：`FinderViewActionResult`、`RefreshFinderWindowOptions`、`ToggleFinderHiddenFilesOptions`，见 `electron/src/main/finder-view-actions.ts:8`

**接入方式**
- 主进程后续可直接 `import { refreshFinderWindows, toggleFinderHiddenFiles } from "./finder-view-actions"`
- `view.refresh` 示例：`await refreshFinderWindows({ directoryPath, activateFinder: true })`
- `view.toggle-hidden` 示例：`await toggleFinderHiddenFiles({ directoryPath, desiredVisible: true })`
- 两个函数都返回统一结果：`{ ok, changed, message, errorMessage, details }`，适合直接做 toast / log / result window 提示
